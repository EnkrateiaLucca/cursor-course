# Section 07: Diagrams & Design

**Slides:** 37–45 | **Duration:** ~30 min | **Break after:** Yes

## Learning Goals

- Generate Mermaid diagrams in Cursor for architecture visualization
- Use the sketch/screenshot → code workflow (Figma or hand-drawn to implementation)
- Understand how to iterate on design with AI assistance
- Create documentation-ready diagrams from your codebase

## Prerequisites

- AI Quiz App from Section 05 as the reference project
- Mermaid preview extension (optional) or use mermaid.live

## Key Talking Points

- Mermaid diagrams are text-based — perfect for version control and AI generation
- Architecture diagrams help you think before building (and document afterward)
- The sketch → code workflow: take a photo/screenshot of a design → paste in Cursor → get code
- AI can iterate on designs: "make this more spacious" or "match this color scheme"
