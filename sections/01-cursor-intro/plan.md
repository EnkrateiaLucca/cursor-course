# Goal?
Goal is to build an html app that can take in 
images and lay them out into a nice pdf layout
for saving or printing.

# Project Overview?
Simple html app that you can upload images
and then have a simple interface to lay them out 
nicely into a pdf with a save pdf button.

# Tech Stack?
Pure html file (javascript + css should be bundled inside).

# Concerns/Constraints
File types it accepts: .jpg, .png, .jpeg, .heic, .webp, pretty much anything that's an image.

# NOtes during iteration

- Extra features we wanted like: dark mode...
- UX: more dynamic interface for moving images around and placing them in the pdf layout
