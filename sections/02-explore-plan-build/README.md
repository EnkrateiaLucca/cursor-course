# Section 02: Explore-Plan-Build Framework

**Slides:** 7–11 | **Duration:** ~45 min | **Break after:** No

## Learning Goals

- Understand the Explore → Plan → Build framework for AI-assisted development
- Master the @ symbol for context navigation (@file, @folder, @codebase, @docs, @web)
- Create a context file that serves as the project's "north star"
- Build a simple quiz app (no AI) on McKay's template to practice the framework

## Prerequisites

- McKay's App Template cloned and running locally
- Cursor IDE with the template open

## Key Talking Points

- Don't jump straight to code — EXPLORE the codebase first, PLAN your approach, then BUILD
- The @ symbol is your context navigator: @file for precision, @codebase for discovery
- A context file saves you from repeating yourself in every prompt
- The two-stage workflow: ChatGPT/Claude for planning → Cursor for implementation
- More context != better results — curate what the AI sees
